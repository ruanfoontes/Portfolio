#  Portfólio - Ruan Fontes

Este repositório contém o código do meu portfólio pessoal, desenvolvido com o objetivo de apresentar meus projetos, habilidades e evolução como desenvolvedor.

O portfólio foi criado para reunir alguns dos projetos que desenvolvi e demonstrar minhas habilidades em desenvolvimento web. Nele é possível conhecer um pouco mais sobre mim, minhas tecnologias e alguns trabalhos realizados.

##  Tecnologias Utilizadas

* HTML5
* CSS3
* Git
* GitHub

##  Seções do Portfólio

* **Início** – Apresentação e foto de perfil
* **Sobre** – Informações sobre minha trajetória e habilidades
* **Projetos** – Alguns projetos desenvolvidos por mim

Este portfólio será atualizado sempre que necessário, incluindo novos projetos, melhorias no código, ajustes no design e novas funcionalidades conforme minha evolução como desenvolvedor.
